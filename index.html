Music Controller and FFT Display

Team Members: Annie Luo, Dimitry Jean-Laurent, Emmy Perez, Shyam Patel

Project Description

For this project, we developed a GUI that allows the user to make a song selection based upon a library of music on an SD card and then based upon the song selected the mbed will perform a FFT analysis and output this to an LED display.

Components

  • Speaker
  • H-Bridge Amplifier
  • Potentiometer
  • SD Card Reader
  • Adafruit LuckyLight 8x8 LED Display
  • mbed LPC1768
